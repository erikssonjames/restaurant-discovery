# QA checklist

## Keyboard

- Tab through the header, filters, cards, tabs, and footer.
- Confirm every interactive element has a visible focus state.
- Confirm the skip link moves focus to the main content.
- Open and close the mobile navigation using the keyboard.
- Use the search form without a mouse.
- Confirm focus does not become trapped.

## Responsive layout

Check these widths:

- 320px
- 375px
- 768px
- 1024px
- 1440px

Confirm there is no horizontal scrolling, clipped text, overlapping content, or unusable controls.

## Browsers

Test the main routes in:

- Chrome
- Edge
- Firefox
- Safari when available

## Content

- Check restaurant images and alt text.
- Check city and cuisine links.
- Check menus, prices, addresses, and phone links.
- Check empty search results.
- Check loading, error, and not-found pages.
- Check sitemap.xml and robots.txt.

## Accessibility

- Test with Windows Narrator or NVDA.
- Confirm headings follow a logical order.
- Confirm form fields have labels.
- Confirm dialogs announce their title.
- Confirm page content remains usable at 200% zoom.
